## Future LwM2M Test Events

OMA celebrates Specification Validation Events (SVE) aka "TestFest" events with the aim to verify the consistency of OMA Technical Specifications. 
Learn more about OMA TestFest events [_here_](https://lwm2m.openmobilealliance.org/testfests/).
